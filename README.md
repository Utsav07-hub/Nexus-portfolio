## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Utsav07-hub/Nexus-portfolio.git

Move into the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
